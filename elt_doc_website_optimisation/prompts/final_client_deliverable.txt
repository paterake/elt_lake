**CRITICAL - Security Headers**:

**Python Finding**:
- ⚠ Missing: HSTS, X-Content-Type-Options, X-Frame-Options, CSP

**Your Deep Analysis + Fix Code**:
```
SEVERITY: HIGH
CVSS Score: 6.5/10
Exploitability: EASY
Business Impact: Data breach risk, SEO impact, user trust

Missing Headers & Risks:
1. Strict-Transport-Security (HSTS)
   - Risk: Protocol downgrade attacks
   - Attack: Forces HTTP, intercepts data

2. X-Content-Type-Options
   - Risk: MIME-type sniffing attacks
   - Attack: Malicious files executed as scripts

3. X-Frame-Options
   - Risk: Clickjacking attacks
   - Attack: Site embedded in malicious iframe

4. Content-Security-Policy (CSP)
   - Risk: XSS attacks
   - Attack: Malicious scripts from external sources

FIX - Add to .htaccess (Apache):
<IfModule mod_headers.c>
  Header set Strict-Transport-Security "max-age=31536000; includeSubDomains"
  Header set X-Content-Type-Options "nosniff"
  Header set X-Frame-Options "SAMEORIGIN"
  Header set Content-Security-Policy "default-src 'self'; script-src 'self' 'unsafe-inline'; style-src 'self' 'unsafe-inline'; img-src 'self' data: https:;"
</IfModule>

Alternative: Add via hosting control panel or security plugin

Effort: 1-2 hours
Cost: £75-150 (developer time)
Risk if not fixed: HIGH (exploitable today)
Priority: P1 (fix within 24 hours)
Testing: Use securityheaders.com after implementation
```

## Style Guidelines

### Writing Style
- **Specific**: Use numbers, names, versions
- **Evidence-based**: Every finding has evidence
- **Actionable**: Every problem has a solution with effort/cost estimate
- **Professional**: Constructive, not alarmist

### Evidence Format
```
✓ Finding description [SEVERITY]
    Evidence: [specific data]
    Analysis: [LLM context and business impact]
    Fix: [specific code or steps]
    Effort: [hours]
    Cost: [£]
    Priority: [P1/P2/P3]
```

### Avoid
- ❌ "Manual review required" (when you HAVE data)
- ❌ "Consider improving" (vague)
- ❌ "Some issues found" (which ones?)

### Use Instead
- ✅ "3 plugins without version info - verify in wp-admin"
- ✅ "Reduce server response time from 668ms to <200ms"
- ✅ "Missing security headers: HSTS, X-Content-Type-Options, X-Frame-Options, CSP"
